# Design a Set Class

Support the following Public API:

Properties:
- `size : int`

Methods:
- `add(item : T) : void`
- `clear() : void`
- `contains(item : T) : bool`
- `get(item : T) : T`
- `remove(item : T) : void`
- `toString() : string`
- `union(other : T) : void`
