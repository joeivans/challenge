# Recursion-1 Factorial
- Learning
- CodingBat

Given `n`, an integer from 1 to 12, return the factorial.

**Example 1:**
```
Input: 1
Output: 1
```

**Example 2:**
```
Input: 2
Output: 2
```

**Example 3:**
```
Input: 3
Output: 6
```

**Example 4:**
```
Input: 4
Output: 24
```

**Example 5:**
```
Input: 5
Output: 120
```

**Example 6:**
```
Input: 6
Output: 720
```

**Example 7:**
```
Input: 7
Output: 5040
```

**Example 8:**
```
Input: 8
Output: 40320
```

**Example 9:**
```
Input: 12
Output: 479001600
```

**Constraints:**
- `1 <= n <= 12`
- No loops allowed.
