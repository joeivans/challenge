# Recursion-1 Fibonacci
- Learning
- CodingBat

Given `n`, return the fibonacci number.

**Example 1:**
```
Input: 0
Output: 0
```

**Example 2:**
```
Input: 1
Output: 1
```

**Example 3:**
```
Input: 2
Output: 1
```

**Example 4:**
```
Input: 3
Output: 2
```

**Constraints:**
- `0 <= n <= 100`
- No loops allowed.
- No multiplication allowed.
