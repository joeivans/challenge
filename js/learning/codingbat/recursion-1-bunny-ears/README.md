# Recursion-1 Bunny Ears
- Learning
- CodingBat

Given `bunnies`, an integer that counts the number of bunnies, return the count
of bunny ears using recursion.

Each bunny has `2` ears.

**Example 1:**
```
Input: 0
Output: 0
```

**Example 2:**
```
Input: 1
Output: 2
```

**Example 3:**
```
Input: 2
Output: 4
```

**Constraints:**
- `0 <= bunnies <= 1000`
- No loops allowed.
- No multiplication allowed.
