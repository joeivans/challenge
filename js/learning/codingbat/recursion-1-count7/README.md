# Recursion-1 Count7
- Learning
- CodingBat

Given `n`, a non-negative integer, return the count of `7`'s in a number.

**Example 1:**
```
Input: 717
Output: 2

There are 2 sevens in the number 717.
```

**Example 2:**
```
Input: 7
Output: 1
```

**Example 3:**
```
Input: 123
Output: 0
```

**Constraints:**
- `0 <= n`
- No loops allowed.
- No multiplication allowed.
- Use recursion.
