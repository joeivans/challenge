# Remove Linked List Tail
- Learning

Given a linked list `head`, remove the last node (`tail`) if it exists.

**Example 1:**
```
Input: head = [1, 2, 3]
Output: [1, 2]
```

**Example 2:**
```
Input: head = [1, 2]
Output: [1]
```

**Example 3:**
```
Input: head = [1]
Output: []
```

**Example 4:**
```
Input: head = []
Output: []
```

**Constraints:**
- `0 <= number of nodes in head <= 1000`
- `-1000 <= Node.val <= 1000`
